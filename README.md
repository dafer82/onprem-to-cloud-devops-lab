
# Laboratorio de DevOps Híbrido – De Infraestructura On-Premise a Azure Cloud

## Resumen del Proyecto
Este repositorio documenta una experiencia práctica en la construcción de una infraestructura híbrida completa, conectando entornos tradicionales on-premise con servicios modernos de nube en Azure.

El objetivo del laboratorio es integrar sistemas heredados con plataformas cloud aplicando buenas prácticas de infraestructura, automatización y DevOps.

## Estructura del Laboratorio
El proyecto se organiza en las siguientes fases:

- **01-Arquitectura**  
  Decisiones técnicas, diseño de la topología de red y estrategia de virtualización.

- **02-Laboratorio Local (On-Premise)**  
  Configuración del entorno local y servicios principales:
  Active Directory, redes, virtualización y Linux.

- **03-Resolución de Problemas (Troubleshooting)**  
  Documentación de problemas reales y sus soluciones, incluyendo conflictos entre Hyper-V y VirtualBox, DNS y escenarios comunes de infraestructura.

- **04-DevOps**  
  Implementación de control de versiones con Git, contenedores, Infraestructura como Código (IaC) y pipelines de integración y despliegue continuo.

- **05-Híbrido Azure**  
  Integración con la nube de Azure, sincronización de identidades y estrategia de modernización híbrida.

## Enfoque Técnico Clave
- **Modernización de Sistemas Legados**  
  Integración de entornos tradicionales (COBOL / Mainframe) con prácticas modernas de automatización y pipelines DevOps.

- **Infraestructura como Código (IaC)**  
  Automatización de despliegues para garantizar consistencia, repetibilidad y control de cambios.

- **Redes Híbridas**  
  Diseño e implementación de comunicación segura entre infraestructura local y servicios en la nube.

## Objetivos del Proyecto
- Diseñar y desplegar un entorno corporativo completo on-premise.
- Aplicar principios DevOps tanto en sistemas legados como cloud-native.
- Construir una estrategia híbrida de identidad y automatización alineada con las mejores prácticas de Azure.

## Estado del Proyecto
Este laboratorio se encuentra en desarrollo activo y en evolución continua como parte de un recorrido práctico de formación en infraestructura, DevOps y entornos híbridos On-Premise + Cloud.
