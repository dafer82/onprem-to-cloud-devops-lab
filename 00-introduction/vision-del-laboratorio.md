# Visión del laboratorio

## Objetivo
Construir un laboratorio de aprendizaje que conecte entornos on-premise
tradicionales (Active Directory, Windows Server, redes) con prácticas modernas de DevOps y Cloud (Azure, CI/CD, IaC).

## Motivación
Este laboratorio no busca únicamente "hacer funcionar" tecnologías sino 
entender los errores, limitaciones y decisiones de arquitectura que aparecen en entornos reales.

## Alcance
- Active Directory on-premise
- Virtualización con VirtualBox
- Automatización y DevOps
- Integración futura con Azure
