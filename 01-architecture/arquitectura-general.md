# Arquitectura general del laboratorio

## Visión
El laboratorio simula una pequeña infraestructura corporativa on-premise con proyección a un entorno híbrido cloud.

## Componentes
- Host físico: Windows 11 Pro
- Virtualización: VirtualBox
- Servidor Windows: Windows Server 2022
- Servidor Linux: Debian (base estable para contenedores y automatización)

## Objetivo arquitectónico
Seperara responsabilidades, mantener reproducibilidad y permitir evolución hacia DevOps y Cloud.
